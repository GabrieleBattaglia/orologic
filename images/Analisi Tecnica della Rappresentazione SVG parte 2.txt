i parametri predefiniti di python-chess richiedono due approcci strutturali separati per manipolare i colori della scacchiera e quelli dei pezzi. La libreria non gestisce il colore dei pezzi tramite dizionario nativo.
Ecco i fatti operativi e le chiavi esatte da utilizzare.
Dizionario colors (Case, bordi e coordinate)
Il parametro colors della funzione chess.svg.board() accetta un dizionario Python in cui le chiavi sono stringhe fisse e i valori sono codici colore esadecimali (es. "#ffce9e" per colori opachi o "#15781b80" per colori con opacità). Le chiavi che ti servono sono:
• 
"square light": colore delle case chiare.
• 
"square dark": colore delle case scure.
• 
"margin": colore di sfondo del bordo della scacchiera che ospita le coordinate.
• 
"coord": colore del testo alfanumerico delle coordinate.
• 
"inner border": colore della linea di delimitazione interna tra le case e il margine.
• 
"outer border": colore del perimetro esterno al margine.
Parametro style (Pezzi, bordi, ombreggiature)
I pezzi generati dalla libreria sono elementi grafici SVG di tipo <path> assegnati a classi CSS specifiche (es. .white.pawn, .black.king). Per alterare i loro colori, bordi o aggiungere ombreggiature, è obbligatorio iniettare un foglio di stile CSS personalizzato tramite il parametro style della funzione board().
È tassativo l'impiego del tag !important su ogni attributo CSS per forzare la sovrascrittura degli stili inline predefiniti, altrimenti elementi composti come i cavalli manterranno i pigmenti di default.
Le proprietà CSS per il tuo scopo sono:
• 
fill: per il colore di riempimento principale del pezzo.
• 
stroke: per il colore del bordo del pezzo.
• 
stroke-width: per lo spessore del bordo.
• 
filter: drop-shadow(...): per creare ombreggiature.
Ecco l'implementazione del codice per applicare queste direttive, redatto secondo i tuoi standard di indentazione e compattezza:
Python
Copia codice
import chess
import chess.svg
board = chess.Board()
board_colors = {
    "square light": "#f0d9b5",
    "square dark": "#b58863",
    "margin": "#333333",
    "coord": "#ffffff",
    "inner border": "#ffffff",
    "outer border": "#000000"
}
custom_css = """
.white.pawn path, .white.knight path, .white.bishop path, .white.rook path, .white.queen path, .white.king path {
    fill: #e6e6e6 !important;
    stroke: #333333 !important;
    stroke-width: 1px !important;
    filter: drop-shadow(3px 3px 2px rgba(0,0,0,0.6)) !important;
}
.black.pawn path, .black.knight path, .black.bishop path, .black.rook path, .black.queen path, .black.king path {
    fill: #222222 !important;
    stroke: #888888 !important;
    stroke-width: 1px !important;
    filter: drop-shadow(3px 3px 2px rgba(0,0,0,0.8)) !important;
}
"""
svg_data = chess.svg.board(board, colors=board_colors, style=custom_css, size=600)
with open("scacchiera.svg", "w") as f:
    f.write(svg_data)
