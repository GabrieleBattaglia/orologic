Orologic User Guide (Version 4.0.0)

Language Note: This guide is currently available only in English. We encourage users to utilize the many excellent online translation tools to read it in their native language.

1. Introduction

Welcome to Orologic, a fully accessible chess clock and game analysis tool designed with visually impaired users in mind. Orologic allows you to play timed chess games, save them in PGN format, and perform detailed analysis using a UCI chess engine.

1.1. Core Principles

  • Accessibility First: Every feature is designed to be used with a screen reader. The hierarchical structure of this guide is a reflection of the program's design.
  • Keyboard Navigation: All interactions are performed via the keyboard.
  • Standalone: Orologic is a single executable file. No complex installation is required.

1.2. Navigating Menus: The Orologic Way

Orologic uses a unique and efficient menu selection system. When presented with a list of choices, you don't need to type the full command. Instead, you can type the first few letters of your desired option.

  • As you type, Orologic filters the list of available options to show only those that start with the characters you've entered.
  • If your input narrows the choices down to a single option, that option is automatically selected.
  • If you've typed a sequence of characters and there are still multiple matching options, pressing the Enter key will select the option that is an exact match for what you typed.

For example, in the main menu, you have options like crea, elimina, gioca, etc. If you type c, the only matching option is crea, so it will be selected instantly. If you type e, both elimina and the exit command . (which is not shown but available) might be considered, so you might need to type more letters like el to select elimina.

2. Main Menu Options

When you start Orologic, you are greeted with a welcome screen. Typing ? and pressing Enter will display the main menu options. Here is a breakdown of each one.

2.1. Play a Game (gioca)

This is the core function for playing a timed chess game against an opponent.

2.1.1. Starting a Game

  1. Select a Clock: You will first be prompted to choose a pre-configured clock. If you haven't created one, you should do so first using the crea option.
  2. Game Mode: You can choose between a Standard game or a Fischer Random (Chess960) game.
    • For Fischer Random, you can either enter a specific 8-character piece sequence (e.g., RNBQKBNR), type ? for a random position, or type . to cancel.
  3. Player Information: You will be asked to enter names and Elo ratings for the White and Black players, as well as event details. The program saves your last entries as defaults for the next game.

2.1.2. During the Game: Making Moves

Moves are entered using Standard Algebraic Notation (SAN). For example:

  • e4 for a pawn move.
  • Nf3 for a knight move.
  • O-O for short castling and O-O-O for long castling.
  • dxe5 for a capture.
  • e8=Q for a promotion.

You can also add annotations directly to your move:

  • ! (good move), ? (bad move), !! (brilliant move), ?? (blunder), !? (interesting move), ?! (dubious move). Example: Nf3!
  • = to signify a draw offer. Example: Qd2=

2.1.3. In-Game Commands (Dot Commands)

While playing, you can enter special commands that start with a period (.). These commands allow you to manage the clock, check game status, and more.

Command Description
.1Show White's remaining time.
.2Show Black's remaining time.
.3Show both clocks.
.4Compare remaining times and state the advantage.
.5Report which clock is running or the duration of the pause.
.lDisplay the list of moves played.
.mShow the material value still on the board for each player.
.pPause or resume the clocks.
.qUndo the last move (only available when the game is paused).
.b+Add time to White's clock (e.g., .b+60 to add 60 seconds). Must be paused.
.b-Subtract time from White's clock. Must be paused.
.n+Add time to Black's clock. Must be paused.
.n-Subtract time from Black's clock. Must be paused.
.sDisplay the current board state as text.
.c [comment]Add a comment to the current move (e.g., .c This is a key moment).
.1-0Assign victory to White and end the game.
.0-1Assign victory to Black and end the game.
.1/2Assign a draw and end the game.
.*Assign an undefined result and end the game.
.?Display this list of commands.

2.1.4. In-Game Board Inspection Commands

You can also inspect the board without it being your turn to move.

Command Description
/[column]Shows the pieces on the top-right diagonal starting from the base of the given column (e.g., /c for the c1-h6 diagonal).
\[column]Shows the pieces on the top-left diagonal starting from the base of the given column (e.g., \f for the f1-a6 diagonal).
-[column|rank|square]Shows the pieces on a specific column, rank, or square (e.g., -e, -4, -e4).
,[PieceSymbol]Shows the position(s) of the specified piece. Use uppercase for White and lowercase for Black (e.g., ,Q for White's Queen, ,n for Black's knights).

2.1.5. End of Game

The game ends automatically on checkmate, stalemate, or other draw conditions (insufficient material, 75-move rule, etc.). When the game is over, Orologic will:

  1. Announce the result.
  2. Save the game as a .pgn file.
  3. Save a human-readable text summary as a .txt file.
  4. Copy the PGN to your clipboard.
  5. Offer to immediately analyze the game if it has at least 4 full moves.

2.2. Analyze a Game (analizza)

This powerful mode allows you to review a game, add comments, and get insights from a chess engine.

2.2.1. Starting Analysis

To start, select analizza from the main menu. The program will attempt to load a PGN from your clipboard. If your clipboard contains multiple games, you will be presented with a menu to choose which one to analyze.

2.2.2. Analysis Mode Commands

Once in analysis mode, you navigate the game and interact with the engine using single-letter commands. The prompt will show you the current move. Press ? at any time to see this list.

Command Description
Navigation
aGo to the start of the game (or the parent node if in a variation).
sGo back one move.
dGo forward one move.
fGo to the end of the current line.
gSelect the previous variation branch.
hSelect the next variation branch.
kGo to a specific move number.
Engine Analysis
qCalculate and show the best move in the current position.
wCalculate and show the best line (sequence of moves).
eShow multiple lines of analysis and enter "Smart Inspection Mode".
rCalculate and show the centipawn evaluation for the current position.
tShow the Win/Draw/Loss percentages for the current position.
PGN and Comments
lLoad a new PGN from the clipboard.
zInsert the engine's best line as a new variation in the PGN.
xInsert the engine's best move as a comment in the PGN.
cAdd a custom text comment to the current move.
vInsert the centipawn evaluation as a comment in the PGN.
nDelete the comment on the current move.
bToggle automatic reading of comments on/off.
Information & Settings
jRead the PGN headers of the game.
yShow the material balance.
uDisplay the current board state as text.
iSet the analysis time (in seconds) for the engine.
oSet the number of analysis lines (MultiPV) for the engine to show.
Exit
.Exit analysis mode. If you made changes, you will be prompted to save the PGN to a new file.

2.2.3. Smart Inspection Mode

After using the e command to display multiple analysis lines, you can choose to inspect one of them in "Smart Mode". This lets you navigate through a proposed line of play move by move. The commands are simple:

  • d: Go to the next move in the line.
  • s: Go to the previous move in the line.
  • r: Re-evaluate the current position in the line.
  • .: Exit Smart Inspection Mode.
  • ?: Show this list of commands.

2.3. Configuration

Orologic is highly customizable. These main menu options allow you to tailor it to your needs.

2.3.1. Clock Management (crea, vedi, elimina)

  • crea: This walks you through creating a new chess clock. You can define multiple time control phases, each with a base time, an increment per move, and a number of moves to complete in that phase. You can also set audible alarms that trigger when the clock reaches a certain time (e.g., 5 minutes).
  • vedi: This displays a list of all your saved clocks and their configurations.
  • elimina: This allows you to delete a clock you no longer need.

2.3.2. Chess Engine Setup (motore)

This is a crucial step for using the analysis features. Here you must provide the path to a UCI-compatible chess engine (like Stockfish). You will be asked for:

  • The full path to the engine's folder.
  • The exact filename of the engine's executable.
  • Configuration options like Hash size (in MB), number of CPU cores to use, and skill level.

2.3.3. Customizing Language and Names (nomi)

This option lets you personalize the descriptive language used by the program. You can change:

  • The names of the pieces (e.g., change "knight" to "horse").
  • The names of the columns (e.g., change "Ancona" to "Apple").
  • The verbs and adjectives used to describe moves, making the output feel more natural to you.

2.3.4. Default PGN Headers (pgn)

Here you can set the default information that will be used in the PGN files for your games, such as your name for White and Black, your Elo, and the default event name and location.

2.3.5. Adjusting Audio Volume (volume)

This simple option allows you to set the volume for the application's sound effects, from 0 (silent) to 100 (maximum).

2.4. Open Manual (manuale)

This command opens this user guide in your default web browser.

2.5. Exit Application (.)

Typing . and pressing Enter from the main menu will close Orologic.
